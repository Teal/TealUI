    ☍正则表达式扩展util/regexp

    提供 JavaScript 内置对象 RegExp 的扩展接口。

    另参考
    • 常用正则表达式库
    • 正则表达式加强

    ☍API

    函数 描述
    ⅏ ⮞ parse(value, ...)(value:string, flags?:string):RegExp

    解析字符串为正则表达式。

    参数 类型 描述 默认值
    value* string

    要解析的字符串。字符串的正则元字符将匹配字符本身。

    		 —
    flags string | undefined

    正则表达式标记。可以是 "gmiu" 中任意字符组合。

    		 —

    返回值

    类型：RegExp

    返回一个新正则表达式对象。

    示例

    ❐parse("\\s") // /\s/

    解析字符串为正则表达式。
    ⅏ ⮞ isRegExp(obj)(obj:any):boolean

    判断对象是否是正则表达式。

    参数 类型 描述 默认值
    obj* any

    要判断的对象。

    		 —

    返回值

    类型：boolean

    如果对象是正则表达式则返回 true，否则返回 false。

    示例

    ❐isRegExp(/a/) // true
    ❐isRegExp("/a/") // false

    判断对象是否是正则表达式。
    ⅏ ⮞ join(...)(...regexps:RegExp[]):RegExp

    合并多个正则表达式并返回匹配其中任一项的新正则表达式。

    参数 类型 描述 默认值
    regexps RegExp[]

    要合并的所有正则表达式。

    		 —

    返回值

    类型：RegExp

    返回一个新正则表达式对象。

    示例

    ❐join(/a/, /b/) // /a|b/

    合并多个正则表达式并返回匹配其中任一项的新正则表达式。
    ⅏ ⮞ fromWildcard(value)(value:string):RegExp

    将通配符转为等效正则表达式。

    参数 类型 描述 默认值
    value* string

    要转换的通配符表达式。其中可使用以下符号：

    • *：匹配任意个字符。
    • ?：匹配一个字符。
    		 —

    返回值

    类型：RegExp

    返回转换后的等效正则表达式。

    示例

    ❐fromWildcard("a*b").test("ab") // true
    ❐fromWildcard("a*b").test("acb") // true
    ❐fromWildcard("a*b").test("acbd") // false

    将通配符转为等效正则表达式。
