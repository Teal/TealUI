本组件提供了网页开发时经常使用的 css 工具类。
色系
默认: 正文 链接
x-hint: 提示 提示链接
x-highlight: 强调 重要链接
x-success: 成功 成功链接
x-warning: 警告 警告链接
x-error: 失败 失败链接
x-info: 信息 信息链接
位置
x-left: 左浮动
x-center: 居中
x-right: 右浮动
x-middle: 文字垂直居中
x-clear: 清除浮动
显示
x-inline: 内联
x-block: 块
x-inline-block: 内联块
x-hide: 隐藏元素（display:none)
定位
x-relative: 相对定位
x-absolute: 绝对定位
杂项
x-plain: 重置 margin=0, padding=0
x-row: margin-bottom=10px
x-blank: 空行（10px高）
x-break-word: 强制文本换行，避免长单词不换行
x-debug: 调试突出
