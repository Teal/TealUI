本组件提供了网页开发时经常使用的 css 工具类。
色系
默认: 正文 链接
ui-hint: 提示 提示链接
ui-highlight: 强调 重要链接
ui-success: 成功 成功链接
ui-warning: 警告 警告链接
ui-error: 失败 失败链接
ui-info: 信息 信息链接
位置
ui-left: 左浮动
ui-center: 居中
ui-right: 右浮动
ui-middle: 文字垂直居中
ui-clear: 清除浮动
显示
ui-inline: 内联
ui-block: 块
ui-inline-block: 内联块
ui-hide: 隐藏元素（display:none)
定位
ui-relative: 相对定位
ui-absolute: 绝对定位
杂项
ui-plain: 重置 margin=0, padding=0
ui-row: margin-bottom=10px
ui-blank: 空行（10px高）
ui-break-word: 强制文本换行，避免长单词不换行
ui-debug: 调试突出
