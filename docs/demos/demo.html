基础结构

使用 HTML5 DocType

CSS Reset

jPlusUI 重置了全局样式，以让中文更美观，并增强浏览器兼容性。重置包括了：

  • 设置默认字体和颜色
  • 重置 body、img、ul 和 table 以方便布局
  • 重置一些标签（包括HTML5）以让浏览器显示相同的效果。

所有的重置样式均由 Base 组件 提供。
